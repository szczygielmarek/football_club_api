## Description

Football Club REST API

Based on: <br>
[Nest](https://github.com/nestjs/nest) - Node.js framework. <br>
[Winston](https://github.com/winstonjs/winston) - Logging library with support for multiple transports.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Documentation

### OpenAPI (Swagger)
While the application is running, open your browser and navigate to http://localhost:3000/api. You should see the Swagger UI.

### TypeDoc

```bash
# typedoc
$ npm run docs
```
The code documentation will be built in the folder /docs
