## Description

Football Club REST API

Based on: <br>
[Nest](https://github.com/nestjs/nest) - Node.js framework. <br>
[Winston](https://github.com/winstonjs/winston) - Logging library with support for multiple transports.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
